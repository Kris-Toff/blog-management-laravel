## Technical Assessment for Full Stack Developer

### Setup instructions
- git clone https://github.com/Kris-Toff/blog-management-quasar.git
- composer install
- Setup .env
  - Database name: blog_management_backend
  - add APP_QUASAR_URL and use quasar dev url (e.g. http://localhost:9000)
- run php artisan migrate
- run php artisan serve
